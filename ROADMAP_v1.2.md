# Roadmap v1.2 — Racoon Typper

## Planned Features

### v1.2
- Import profile (full restore from JSON)
- Custom themes (user-defined CSS)
- Additional languages: Spanish, French, German
- Onboarding tutorial (first-run guide)
- Achievement gallery (view all achievements)
- Sound customization (choose sound files)
- Keyboard layout support: Dvorak, Colemak
- Daily goal system (custom WPM/accuracy targets)
- Weekly summary report

### v1.3
- Plugin system (WASM + Native)
- Course editor (create custom lessons)
- Import/export data (Anki, CSV)
- Screenshot generator (SVG → PNG share card)
- Performance benchmark suite

### Post v1.0 (no timeline)
- Multiplayer LAN mode
- Multi-language UI (i18n)
- Adaptive difficulty (ML-based)
- Cloud sync (optional, opt-in)
- Mobile companion app
- Voice narration for lessons
- Eye-tracking integration
